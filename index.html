Welcome to the Disting mk4 Quick Guide. This guide is a cheat sheet for the 97 algorithms included with the Expert Sleepers Disting mk4 (as of firmware update 4.10). Since the bulk of the text of the manual is not repeated here, the Quick Guide is not designed to be your primary source for getting to know your Disting. Once you have a basic understanding of how the algorithms work, however, I hope that you’ll find this page a handy map to the ins, outs and parameter settings of each algorithm.

In addition to distilling the info to the basics, this guide is designed to be mobile-friendly, so you can view it without being tied to your computer. You can save the guide to the homescreen of your mobile device for easy mobile access.

If you find any errors or have any suggestions, please don’t hesitate to send them to damon@rustleworks.com. The code is open sourced under the MIT License. If you’d like to clone it for your own purposes, you can find the source at Github.

Happy Bleeping!

LIVE mode CV Control

Starting with firmware 1.2, THIS or THAT can be assigned alternate behaviors in LIVE mode. Enter the selector to change THIS/THAT modes by pressing record while inserting a cable into THIS or THAT, then select the new mode by pressing the button indicated below. To mix/match modes, press and hold your THIS selection then press your THAT selection.

dub

toggle recording and level (default)
select with play
THIS
input: trigger
toggle recording (punch in/out)
while playing
THAT
input: 0 to +5v
playback level (5v is quieter)
while recording
THAT
input: 0 to -5v
overdub (0v) to overwrite (-5v) - previous material not heard
input: 0 to +5v
overdub (0v) to overwrite (5v) - previous material is heard

transport

cues and speed (identical to NAV mode)
select with ▲
THIS
input: -5v to +5v
when playhead reaches a cue point, jumps to the cue indicated by THIS voltage - all cue points are evenly distributed
THAT
input: -5v to +5v
tape speed - fast backwards (-5v) to fast forwards (+5v)

signal

location ramp output and boundry trigger
select with loop
THIS
output: -5v to +5v
outputs voltage according to loop position (-5V start to +5v end)
THAT
output: trigger
sends a trigger whenever the play head hits a cue point

caption

record CV
select with record
THIS
input: -5v to +5v
CV to be recorded
THAT
output: -5v to +5v
recorded CV output

sample

jump cues and v/octave pitch shift (similar to CUE mode)
select with ▼
THIS
input: trigger
go to the most recently passed cue immediately - attaching a parallel static voltage (-1V to 1V) allows it to jump forward or backward by a single cue
THAT
input: -5v to +5v
v/octave tape speed
